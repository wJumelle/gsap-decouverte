Présentation du principe

L'idée est simple "stagger" signifie échelonnement, ce qui permet tout simplement d'exécuter des animation les unes après les autres, en y ajoutant un délai (en seconde).

On verra qu'il existe différents moyen de gérer cet échelonnement mais le principe de base est assez simple, il faut avoir une target qui cible plusieurs éléments du DOM et ajouter la propriété objet stagger.

const tween_1 = gsap.to('.animation--1 .square', { paused: true, x: 600, stagger: 0.25 });

document.getElementById('anim_1_button').addEventListener('click', () => {
    tween_1.restart();
});
